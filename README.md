# Stack Overflow Developer Survey Analysis

## 📌 Project Overview
Analysis of the Stack Overflow Developer Survey dataset using Python and Pandas to explore developer trends, compensation, and technology usage.

## 🛠 Tools & Technologies
- Python
- Pandas
- Jupyter Notebook
- Descriptive Statistics

## 🔍 Key Tasks
- Loaded and explored large-scale survey dataset
- Handled missing values and validated data completeness
- Calculated mean, median, and mode for work experience
- Analyzed Python usage percentage
- Evaluated compensation by country
- Identified trends among top-paid developers

## 📊 Key Findings
- Python remains one of the most widely used programming languages
- Remote work adoption is significant across regions
- Compensation varies considerably by country

## 🚀 Outcome
Generated structured statistical insights to better understand global developer trends.
